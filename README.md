# Detección de fraude en seguros de automóvil mediante *machine learning*

Este repositorio contiene el código utilizado en el TFM **“Detección de fraude en seguros de automóvil mediante machine learning”**.

## Contenido
- Preparación y limpieza de la base de datos
- Transformación de variables y creación de dummies
- Estimación de modelos de clasificación:
  - Regresión logística
  - Árbol de decisión
  - Random Forest
  - XGBoost
- Evaluación de resultados mediante:
  - *accuracy*
  - *precision*
  - *recall*
  - *F1-score*
  - matriz de confusión
  - curva *Precision-Recall*

## Archivo principal
- `notebook_tfm_fraude.ipynb`

## Dataset utilizado
El análisis se ha realizado a partir del dataset **Vehicle Insurance Fraud Detection**, disponible en Kaggle:

https://www.kaggle.com/datasets/khusheekapoor/vehicle-insurance-fraud-detection

## Entorno de trabajo
El código se ha desarrollado en **Python** y ejecutado en **Google Colab**.

## Nota
El repositorio tiene como finalidad facilitar la trazabilidad y reproducibilidad del análisis empírico desarrollado en el TFM.
